# פרוטוקול מענה מבצעי - שירותי רפואה (מד"א)
## אירוע: נפגעים בזירת שריפה
1. טריאז' (מיון ראשוני): סיווג הנפגעים לפי דרגת דחיפות (אדום, צהוב, ירוק).
2. טיפול בכוויות: הרחקה ממקור החום, קירור האזור הפגוע ומתן חמצן במידת הצורך.
3. פינוי: תעדוף פינוי נפגעים במצב קריטי לבתי חולים ייעודיים (מרכזי טראומה).
4. בטיחות צוות: אין להיכנס לרכב בוער לפני אישור כוחות הכיבוי.
